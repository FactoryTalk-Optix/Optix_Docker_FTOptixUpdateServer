# FactoryTalkOptixRuntimeToolsSetup Folder

Place the following bash installer scripts in this directory before building the Docker image:

- `FTOptixEntitlementCli.Ubuntu_22_x64.X.Y.Z.ZZZ.sh` - FactoryTalk Optix Entitlement CLI installer
- `FTOptixApplicationUpdateService.Ubuntu_22_x64.X.Y.Z.ZZZ.sh` - FactoryTalk Optix Application Update Service installer
